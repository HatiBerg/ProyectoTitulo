*Cosas que podria hacer para el manejo de respuestas:
	- Mejorar la acción para manejar coincidencias aproximadas y respuestas incompletas.
	- Actualiza story.yml para manejar respuestas incompletas para evitar perder el contexto.
	- Añadir una acción para solicitar información adicional en caso de respuesta incompleta.
	- Añadir un intent para las respuestas incompletas (sin sentido, responder con una palabra, falta año, persona, lugar, etc.).
	
Ejemplo de story.yml:

- story: Manejo de respuestas incompletas: rebellion_start
    steps:
      - intent: rebellion_start
      - action: utter_rebellion_start_incomplete # Hacer esto o una custom action
	  
*Añadir un intent para respuestas incorrectas.
